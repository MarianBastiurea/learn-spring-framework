# 🌱 Spring Framework Course – Java

This repository contains the materials and practical examples developed during a comprehensive **Spring Framework** course using Java.

The goal of this course was to understand and apply both fundamental and advanced concepts from the Spring ecosystem to build modern web applications.

---

## 📚 Topics Covered

### 1️⃣ Java Fundamentals – Object Creation
- Object creation in Java
- Constructors and factory methods
- OOP principles
- Object lifecycle management

---

### 2️⃣ Spring Core & Dependency Injection
- IoC (Inversion of Control)
- Dependency Injection (constructor vs field injection)
- Bean lifecycle
- Java-based configuration (`@Configuration`, `@Bean`)
- Component scanning

---

### 3️⃣ Spring Data JPA
- Database configuration
- Entities and relationships (`@Entity`, `@OneToMany`, `@ManyToOne`)
- Repository pattern
- Derived query methods
- JPQL
- Transaction management

---

### 4️⃣ Spring Scheduling (Jobs)
- Creating scheduled tasks
- `@Scheduled`
- Task execution
- Thread pool configuration

---

### 5️⃣ Spring Web Application
- Building REST APIs
- `@RestController`
- `@RequestMapping`
- `@GetMapping`, `@PostMapping`
- DTO pattern
- Validation with `@Valid`
- Global exception handling

---

### 6️⃣ Spring Security
- Authentication and authorization
- Security filter chain configuration
- Role-based access control
- Password encoding

---

### 7️⃣ JWT (JSON Web Token)
- Token generation
- Token validation
- Stateless authentication
- Integrating JWT with Spring Security

---

### 8️⃣ Testing
- Unit testing with JUnit
- Mocking with Mockito
- Controller and service layer testing
- Repository testing

---

## 🛠 Technologies Used

- Java 17+
- Spring Boot
- Spring Data JPA
- Spring Security
- JWT
- H2 / MySQL
- Maven / Gradle
- JUnit
- Mockito

---
🎯 Course Objectives
Understand Spring application architecture
Build and secure a REST API
Use JPA effectively for persistence
Apply unit testing and mocking best practices
Develop a production-ready backend application
